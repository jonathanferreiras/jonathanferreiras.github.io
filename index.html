Johnbot é um projeto pessoal desenvolvido em php usando a API da Betfair, com a finalidade de efetuar apostas esportivas automaticamente.
O bot filtra jogos reais de futebol e analisa estatísticas de sites para efetuar a devida decisão em sua aposta.

O projeto é alterado regularmente em busca de uma estratégia lucrativa.
